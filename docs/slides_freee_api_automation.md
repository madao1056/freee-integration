**会計freee APIで"経理時間"を削る実践ガイド**

## スライド2: こんな状態になっていませんか？

- 毎月の請求書作成に30分〜1時間
- 銀行明細を目視で確認 → 手入力で仕訳
- 経費レシートが溜まってから一気に処理
- 確定申告前に地獄の帳簿整理

**→ 年間50〜100時間が"経理作業"に消えている**

## スライド3: この企画のゴール

- :x: 会計に詳しくなること
- :o: **会計に使う時間と認知コストを削ること**
- 「作業」→「確認」に変える
- 一度仕組みを作れば、毎月回り続ける

---

## スライド4: freee APIとは？

- freee会計の機能をプログラムから操作できる仕組み
- 画面でポチポチやっていることを、コードで一括実行
- **有料プラン（スターター以上）で利用可能**
- API開発者登録自体は無料（Developer Console）

```
[ freee画面で手作業 ]  →  [ APIで自動実行 ]
  1件ずつ入力              一括で処理
  毎回ログイン             コマンド1つ
  30分〜1時間              数秒〜数分
```

---

## スライド5: APIの基本構造（ざっくり）

- エンドポイント: `https://api.freee.co.jp/api/1/`
- 認証: OAuth 2.0（トークンで通信）

| 操作 | メソッド | 例 |
|------|---------|-----|
| 取得 | GET | 取引一覧、勘定科目、口座明細、試算表 |
| 登録 | POST | 取引作成、請求書発行 |
| 更新 | PUT | 取引修正、ステータス変更 |

```
あなたのPC  →  APIリクエスト  →  freee  →  レスポンス（結果）
```

---

## スライド6: どこまで自動化できるのか？

| 業務 | 手動 | API自動化後 |
|------|------|------------|
| 請求書作成 | 画面で1件ずつ入力 | スプレッドシートから一括発行 |
| 入出金確認 | freee画面を毎日開く | 通知が届く / 一覧を自動更新 |
| 仕訳登録 | 1件ずつ手入力 | スプレッドシートから一括登録 |
| 経費処理 | レシート撮影→手入力 | フォルダに入れるだけで自動アップロード |
| 月次確認 | 試算表を目視 | レポートが自動生成される |

## スライド7: 自動化の全体像

```
┌─────────────────────┐
│  Google Sheets      │  ← 入力・確認の「操作画面」
│  （スプレッドシート） │
└────────┬────────────┘
         │
    ↕ 自動化スクリプト ↕
         │
┌────────┴────────────┐
│  freee API          │  ← 会計データの本体
│  （会計・請求書）     │
└────────┬────────────┘
         │
┌────────┴────────────┐
│  Google Drive       │  ← レシート・証憑の保管
└─────────────────────┘
```

**ポイント: スプレッドシートが「操作画面」になる**

## スライド8: デモ① スプレッドシート → freee 取引登録

- スプレッドシートに1行書くだけで仕訳が登録される
- 入力列: 日付 / 勘定科目 / 金額 / 取引先 / 摘要 / 税区分
- コマンド1つで一括インポート
- 処理済みは自動でステータス更新

**画面操作ゼロで仕訳完了**

```
実行コマンド:
$ node main.js sheets:import <スプレッドシートID>
```

---

## スライド9: デモ② freee → スプレッドシート エクスポート

- freeeの全データをスプレッドシートに一括出力
  - 取引一覧（取引先・勘定科目・税区分付き）
  - 損益計算書（PL）
  - 貸借対照表（BS）
- **確定申告前のチェックがスプレッドシート上で完結**
- フィルタ・ソート・関数で自由に分析可能

```
実行コマンド:
$ node main.js sheets:export <スプレッドシートID>
```

---

## スライド10: デモ③ 月次レポート自動生成

- 指定月の経費を自動集計
- 勘定科目別・取引先別の内訳表を自動作成
- 前月比較
- Sheets上でグラフ化も簡単

**「今月いくら使った？」が一瞬でわかる**

```
実行コマンド:
$ node main.js sheets:report <スプレッドシートID> 2026-02
```

---

## スライド11: デモ④ 請求書データの連携

- freeeに登録済みの請求書をスプレッドシートに一括出力
- スプレッドシートから請求データを収入取引として一括登録
- 請求書APIを使えば、発行・更新の自動化も可能

**請求まわりの転記・確認作業をゼロに**

```
実行コマンド:
$ node main.js sheets:invoice <スプレッドシートID> export
```

## スライド12: どれくらい時間が削れるのか？

| 業務 | Before | After | 削減 |
|------|--------|-------|------|
| 仕訳登録（月20件） | 60分 | 5分 | **-55分** |
| 請求書作成（月3件） | 30分 | 5分 | **-25分** |
| 月次確認 | 30分 | 5分 | **-25分** |
| 経費処理 | 30分 | 5分 | **-25分** |
| **月合計** | **2.5時間** | **20分** | **-2時間** |
| **年間** | **30時間** | **4時間** | **-26時間** |

## スライド13: 「でも、エンジニアじゃないと無理では？」

必要なのは:

1. freeeアカウント（有料プラン・スターター以上）
2. Googleアカウント（スプレッドシート用）
3. 初期セットアップ（1回だけ・ガイド付き）

**コードを書く必要はない。用意されたコマンドを実行するだけ。**

対話型セットアップガイドで1ステップずつ案内します。

---

## スライド14: 始め方（3ステップ）

```
Step 1:  freee APIアプリを作成
         → Developer Console で5分

Step 2:  初期設定を実行
         → ガイドに沿って環境構築

Step 3:  コマンドを叩く
         → sheets:export, sheets:import 等
```

**一度作れば、あとは毎月同じコマンドを繰り返すだけ。**
